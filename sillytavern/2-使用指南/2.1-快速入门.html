2.1 快速入门

快速入门

我对这个完全不懂。请告诉我最快、最简单的开始使用 SillyTavern 的方法。 -- 匿名

你可以在几分钟内轻松开始使用 SillyTavern。以下是两种简单的方式：

  • 你可以通过 2.1 快速入门 免费使用。AI Horde 是一个社区驱动的 AI 服务，提供多种 AI 模型的访问。
  • 如果你有 OpenAI 账户或想注册一个账户，可以 2.1 快速入门 开始。

使用 AI Horde 快速入门

  1. 按照 1 安装 并启动 SillyTavern。

  2. 在 SillyTavern 的初始界面中，输入你的人物名字。这个名字将在聊天中使用。

  3. 点击顶部栏的 API 连接按钮。

  4. 输入 AI Horde 的 API 密钥。你可以暂时使用 0000000000，或者从 AI Horde 获取一个免费密钥。

  5. 选择一些 AI 模型使用。可以从顶部的几个模型中选择。你以后可以随时更改。

  6. 关闭 API 连接窗口。在底部的聊天框中输入一条消息，然后按 Enter。

  7. 你的 AI 会在几秒钟内做出回应。你可以继续 聊天。成功了！

使用 OpenAI 快速入门

安装 SillyTavern

按照 1 安装 并启动 SillyTavern。

获取 OpenAI 访问权限

  1. 注册 OpenAI 账号。
  2. 访问 https://platform.openai.com
  3. 点击右上角的账户图标，然后选择 "View API Keys"。
  4. 点击 "Create new secret key"。将密钥复制并妥善保存。请勿分享此密钥，任何拥有此密钥的人都可以使用你的账户消耗 GPT 服务。

配置 SillyTavern 使用你的 API

  1. 在 SillyTavern 的顶部栏中，点击 API 连接。
  2. 在 API 部分，选择 Chat Completion (OpenAI)。
  3. 在 Chat Completion Source 中，选择 OpenAI。
  4. 粘贴你刚才保存的 API 密钥。
  5. 点击连接按钮，确认显示为有效。
  6. 默认情况下，SillyTavern 会使用 GPT-4 Turbo。你可以选择不同的模型，但需要了解定价。

测试你的设置

  1. 在 SillyTavern 的顶部栏中，点击最右侧的角色管理。
  2. 选择一个现有角色，例如 Seraphina。
  3. 在底部的文本框中，写下对 Seraphina 的话，然后按 Enter 或点击发送按钮。

如果一切设置正确，几秒钟后，Seraphina 会做出回应。
