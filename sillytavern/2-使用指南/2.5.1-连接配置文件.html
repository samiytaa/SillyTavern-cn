title: 连接配置文件

连接配置文件

保存连接配置文件，以便快速在不同的 API、模型和格式模板之间切换。当您需要频繁切换不同的 API 连接或配置时，这个功能特别有用，无需在菜单中进行繁琐的查找。

访问连接配置文件

从 SillyTavern 1.12.6 或更高版本开始，此功能作为内置扩展启用，并可以通过 API 连接菜单访问。如果您希望 禁用 它，可以打开扩展面板，点击“管理扩展”，找到连接配置文件，取消勾选“启用”复选框，然后点击“关闭”。

保存内容

连接配置文件会保存以下选择项：

通用设置

  • API 类型、模型和服务器 URL
  • 设置预设

文本生成 API

  • 系统提示及其状态
  • 指令模式状态和模板
  • 上下文模板
  • 分词器

对话生成 API

  • 代理预设

管理连接配置文件

!!! info 提示

配置文件只保存下拉菜单中选择的内容，不涉及底层设置。这意味着在切换到不同配置文件时，您将失去未保存的更改。为了避免这种情况，确保在切换配置文件之前更新所有预设和模板，以免丢失临时更改。

!!!

  • 要保存配置文件，请设置好所有需要的选项，然后点击“创建”按钮。接着，检查设置并为该配置文件提供一个名称。名称应唯一。
  • 要查看所选配置文件的详细信息，点击“信息”按钮。再次点击可隐藏详细信息。
  • 配置文件设置保存在 settings.json 文件中，直到您点击“更新”按钮才会更新保存的配置文件。这意味着，如果您设置了一个配置文件，但没有更新然后切换到另一个配置文件，您将丢失之前的所有更改。
  • 要恢复从已保存配置文件中更改的选项，点击“重新加载”按钮。
  • 要删除配置文件，点击“删除”按钮并确认删除。此操作无法恢复。

斜杠命令

您可以使用以下斜杠命令来管理连接配置文件：

  1. /profile [name] - 如果提供了参数，则切换到指定的配置文件，否则返回当前配置文件的名称。
  2. /profile-create [name] - 将当前设置保存为一个新配置文件，并为其提供名称。
  3. /profile-list - 返回一个 JSON 格式的可用配置文件名称列表。
  4. /profile-get [name] - 获取指定配置文件的详细信息，以 JSON 格式返回。
  5. /profile-update - 使用当前设置更新选中的配置文件。
