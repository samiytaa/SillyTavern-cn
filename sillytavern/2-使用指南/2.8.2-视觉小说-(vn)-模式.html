视觉小说(Visual Novel, VN) 模式

视觉小说 模式是 SillyTavern 中的一个特殊屏幕布局，允许用户与角色进行对话，角色以类似《心跳文学社》、 《Grisaia 的果实》、 《Fate/stay night》等著名视觉小说游戏中的人物立绘（或角色卡图像）方式呈现。

启用视觉小说模式

启用视觉小说模式

在 SillyTavern 中，视觉小说 模式是内置功能，可以通过进入用户设置（用户设置图标）并在无文本阴影下勾选 视觉小说 模式 来启用。

用户设置

禁用视觉小说模式

禁用视觉小说模式与启用的步骤相同。只需取消勾选视觉小说模式，您将返回到普通的聊天界面。

视觉小说 UI

VN 显示

在视觉小说模式下，UI 会略有改变，以便展示角色立绘（或角色卡图像），这些图片通常位于屏幕中央。然而，在多人群聊中，多个角色的立绘会自动分布到合适的位置，避免重叠，如下所示：

启用 MovingUI 的 VN 模式

如果在用户设置中启用了 MovingUI，角色立绘（或角色卡图像）可以自由移动，您可以根据需要将它们放置在屏幕的任何位置。

群体 VN 显示（MovingUI）

获取角色立绘

获取角色立绘可以通过浏览互联网寻找现有的立绘，例如来自《心跳文学社》或《CounterSide》这类视觉小说或游戏中的角色。如果您想要的角色没有现成的立绘，您还有以下几种选择：

  1. 搜索该角色的立绘包或相关链接。
  2. 使用 LoRA 和 Stable Diffusion 创建自己的立绘。
  3. 使用角色卡图像。虽然这不完全是立绘，但至少在屏幕上有些可视内容。请注意，VN 模式中无法使用多个角色卡图像。

VN 扩展功能

Prome 视觉小说扩展

Prome 视觉小说扩展是由 Bronya Rand 和 Prometheus 推出的第三方扩展，它进一步增强了 SillyTavern 中的视觉小说体验。此扩展包括了诸如 Letterbox 模式（将视觉小说 UI 变得更具“电影感”）、 Focus 模式（使角色立绘变暗）、传统 VN 模式（仅显示最后一条消息）等功能，未来还会有更多功能。

要安装 Prome 视觉小说扩展，您可以通过访问 下载扩展与资源 并找到 Prome 视觉小说扩展 来进行安装，或者按照 Prome 视觉小说扩展 GitHub 页面上的安装说明进行操作。调整 Prome 设置可通过 扩展 -> Prome（视觉小说扩展） 或通过 🪄（魔杖）菜单进行。
