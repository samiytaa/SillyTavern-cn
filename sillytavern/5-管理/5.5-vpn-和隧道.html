title: VPN 和隧道

VPN 和隧道

VPN 和隧道是一种安全的方式，让你可以从世界任何地方访问你的家庭网络。本指南将向你展示如何使用 VPN 或隧道从任何地方访问你的 SillyTavern 实例。

方法

1. 使用 自建 VPN。

许多路由器提供了在路由器管理页面上托管 VPN 服务器的功能（主要是 OpenVPN 或 WireGuard）。请参考你的路由器手册来设置 VPN，并将你的设备添加到 VPN 中。一旦连接，直接访问你为 SillyTavern 设置的私有 IP 地址，就可以顺利连接。对于普通用户，尤其是 Windows 用户来说，这种方法较为简便。

2. 使用 Cloudflare Zero Trust。

Cloudflare Zero Trust 是 Cloudflare 提供的一项免费的组织功能，允许你添加最多 50 个用户。这将通过 Cloudflare 代理你的流量，并通过使用 cloudflared 将你的 ST 计算机设置为隧道，你可以像在家中一样连接到 SillyTavern 实例。

需要注意的是，建立隧道后，你还需要向路由器的私有 IP 地址添加路由，并计算 IP CIDR 值，才能通过 Cloudflare Zero Trust 实现全局本地访问。

3. 使用独立的 Cloudflare 或 ngrok 隧道。

与 AI 后端连接的方式类似，你也可以通过 Cloudflare 隧道连接你的 ST 实例，并打开 Cloudflare 隧道页面。但是，每次你想使用 ST 时，你需要复制并粘贴 Cloudflare/NGROK 生成的每个新链接。

4. 使用 Tailscale。

Tailscale 是一个 VPN 提供商，能够实现安全的远程连接到你的计算机。

Tailscale 设置

Tailscale 是一个 VPN 提供商，允许你安全地远程连接到计算机。Tailscale 的开源实现也存在，你也可以使用 Headscale 来托管 Tailscale 服务器，但这超出了本教程的范围。

1. 创建账户

  • 访问 Tailscale 官网 并创建一个新账户。

注意： 对于单人日常使用，Tailscale 永久免费。如果你担心潜在的费用，只需不添加任何支付选项即可。

2. 设置客户端

  • 访问 Tailscale 下载页面，下载客户端/应用到运行 SillyTavern 的设备和你希望远程连接的设备。
  • 在两个设备上使用你之前创建的账户登录。
  • 访问 Tailscale 管理页面，并批准两个设备。
  • 记下两个连接设备的名称。

3. 将设备添加到白名单

  • 按照 管理白名单 IP 地址 的指南，将你要用来远程访问 SillyTavern 的设备的机器名称添加到 SillyTavern 的白名单中。

4. 连接

现在，每当你想从任何地方使用 SillyTavern 时，只需执行以下步骤：

  • 在运行 SillyTavern 的计算机和你想要远程连接的设备上打开 Tailscale。
  • 在想要连接的设备上打开浏览器，访问 http://<运行 SillyTavern 的计算机名称>:8000/。

5. 与朋友共享 SillyTavern 实例（可选）

  • 告诉你的朋友创建他们自己的 Tailscale 账户，并在他们的设备上下载客户端。
  • 访问 Tailscale 管理页面。
  • 在你的计算机上悬停在三个点按钮上，然后点击 "Share..." 或点击三个点按钮并选择 "Sharing settings..."。
  • 取消勾选 "Allow use as an exit node"（除非你希望你的朋友能够通过你的计算机路由他们的所有互联网流量）。
  • 你可以通过电子邮件发送分享链接，或切换到 "Copy share link" 标签，点击大蓝色按钮，然后通过任何方式将链接发送给你的朋友。
  • 你的朋友点击分享链接后，便会看到你的计算机出现在他们的 Tailscale 网络中。
  • 发送给你的朋友与你使用的相同链接，按照上一步骤访问 SillyTavern。

注意： 这将给予你的朋友对你计算机上所有本地服务的完全访问权限，包括 SillyTavern、automatic1111 等等。只有在你完全信任你的朋友时才应进行此操作。
