3 扩展功能

扩展功能

SillyTavern 配备了许多可以启用或禁用的扩展。扩展可以增加新功能、修改现有功能的行为，或为你的 AI 提供额外的内容。你还可以从“下载扩展和资源”菜单中安装更多扩展。

扩展面板

要打开或关闭扩展面板，选择顶部栏中的 扩展。

  • 管理扩展: 启用、禁用或更新扩展
  • 下载扩展和资源: 从 SillyTavern 仓库安装3 扩展功能、角色、声音和背景
  • 扩展更新提醒: 勾选此项以在安装的扩展有更新时收到通知
  • 安装扩展: 从 Git 仓库 URL 导入扩展
警告

使用第三方扩展可能会产生意外副作用，并可能带来安全风险。
在通过 安装扩展 导入扩展之前，请确保你信任该来源。
我们不对任何第三方扩展所导致的损害负责。

内置扩展

这些扩展是 SillyTavern 内置的，无需安装。你可以在扩展面板中启用或禁用它们。

可安装扩展

从 扩展 中的“下载扩展和资产”菜单安装以下扩展。

!!! info 扩展不是附加功能

附加功能项目已于 2024 年 4 月停止。您不需要安装附加功能即可使用扩展。

!!!

3.1 Blip

使角色消息的文本具有可变速度的动画，并在动画过程中播放声音。

3.4 动态音频

为您的聊天添加沉浸式背景音乐和环境音效。

3.5 EmulatorJS

直接在 SillyTavern 聊天中玩复古游戏。

3.9 Live2D

添加对 Live2d 模型的支持。可自定义表情、动画和互动。

3.10 目标功能

为 AI 设置一个目标，以便在聊天过程中进行目标导向。

3.11 基于检索的语音转换 (RVC)

为文本到语音模块添加实时语音克隆功能。

3.12 语音识别

使用浏览器或附加功能将语音转为文本。

3.17 VRM

添加对 VRM 模型的支持。可自定义表情、动画和互动。

3.18 网页搜索

将网页搜索结果添加到 LLM 提示词中。

  • AccuWeather

通过 AccuWeather API 提供天气信息，支持通过斜杠命令或功能工具查询。

  • Chat Top Bar

为聊天窗口添加一个顶部栏，提供快速操作的快捷方式。

  • Chess

与 LLM 一起下国际象棋。

  • Code Runner

允许在聊天中运行 JavaScript 和 STscript 代码块。

  • D&D Dice

为您的掷骰需求提供一套经典的 D&D 七面骰。

  • Duplicate Finder

增加了按相似性分组角色的功能，方便查找重复角色。

  • Emoji Picker

为聊天消息添加一个按钮，快速插入表情符号。

  • Group Greetings

允许设置针对群聊的特定问候语。

  • Group SendAs

为所选的群聊成员快速插入/sendas 命令模板。

  • HypeBot

基于您最近的聊天内容，使用 NovelAI 的 HypeBot 引擎提供个性化建议。需要有效的 NovelAI 订阅。

  • Idle

在您长时间空闲后，增加“空闲提示”，以自然地继续对话。

  • Image Metadata Viewer

查看附加到聊天的放大图像的元数据。

  • LaTeX

在聊天消息中渲染 LaTeX 和 AsciiMath 公式。

  • Mermaid

在 SillyTavern 聊天中添加 Mermaid 图表和流程图渲染。

  • Notebook

提供一个存储笔记的地方，支持富文本格式。

  • Parameter Randomizer

提供随机化 API 设置滑块的功能，每次生成时都可以变化。

  • Prome Visual Novel Extension

增强当前的视觉小说体验，增加更多功能（聚焦模式、信箱模式等）。

  • Prompt Inspector

增加了在将提示词发送到服务器之前，查看和编辑输出提示词的选项。

  • Push Notifications

允许接收聊天消息的推送通知。

  • Quick Persona

在聊天栏中添加一个下拉菜单，用于选择用户角色。

  • RSS

作为斜杠命令或功能工具，通过 RSS 获取最新新闻。

  • Screen Share

当您发送消息时，提供屏幕图像以支持多模态模型。

  • Silence Player

为扩展菜单添加一个静音音频播放器。如果浏览器标签页在后台被关闭时，此功能有助于防止中断。

  • Timelines

为聊天历史记录添加时间轴导航。

  • Variable Viewer

提供一种简便的方式来查看和修改变量。

  • WebLLM

提供一个界面，允许扩展直接在浏览器中使用语言模型。
