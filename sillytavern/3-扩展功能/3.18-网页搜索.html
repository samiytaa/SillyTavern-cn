3.18 网页搜索

Web Search

将网页搜索结果添加到LLM提示词中。

可用的搜索来源

Selenium 插件

需要安装并启用官方服务器插件。

详情请参见 SillyTavern-WebSearch-Selenium。

支持Google和DuckDuckGo引擎。

Extras API

需要在主机上安装websearch模块和Chrome/Firefox浏览器。

支持Google和DuckDuckGo引擎。

SerpApi

需要SerpApi密钥并提供Google搜索访问。

在此获取密钥: https://serpapi.com/dashboard

SearXNG

需要SearXNG实例URL（可以是私有或公共）。使用HTML格式的搜索结果。

了解更多信息: https://docs.searxng.org/

Tavily AI

需要API密钥。获取密钥请访问: https://app.tavily.com/ :icon-lock:

如何使用

  1. 确保你使用的是最新版本的SillyTavern。
  2. 通过“SillyTavern的下载扩展和资源”菜单安装该扩展。
  3. 打开“Web Search”扩展设置，输入你的API密钥或连接到Extras，并启用扩展。
  4. 网页搜索结果将在你聊天时根据需要自动添加到提示词中。只有用户消息会触发搜索。
  5. 要更自然地包含搜索结果，可以将搜索查询用单个反引号包围起来，例如：告诉我有关`瑞安·高斯林的最新电影`的信息，这将产生查询 瑞安·高斯林的最新电影。
  6. 可选地，你可以根据自己的喜好调整设置。

设置

一般设置

  1. 启用 - 开关扩展的启用与禁用。
  2. 源 = 设置搜索结果的来源。
  3. 缓存生命周期 - 搜索结果在提示词中缓存的时间（单位：秒）。默认值 = 一周。

提示词设置

  1. 提示词预算 - 设置插入文本的最大字符数（单位：字符，不是tokens）。一般来说，1个token大约等于3-4个字符，可根据模型的上下文限制进行调整。默认值 = 1500字符。
  2. 插入模板 - 搜索结果插入到提示词中的方式。支持常规宏和特殊宏：{{query}} 表示搜索查询，{{text}} 表示搜索结果。
  3. 插入位置 - 搜索结果插入提示词的位置。可以选择与作者注释一样的选项：插入到聊天中或在系统提示词之前/之后。

搜索激活

  1. 使用功能工具 - 使用功能调用来激活搜索或抓取网页。必须使用支持的聊天完成API，并在AI响应设置中启用。启用时会禁用所有其他激活方法。
  2. 使用反引号 - 启用通过单个反引号包围的关键词来触发搜索。
  3. 使用触发短语 - 启用通过触发短语来激活搜索。触发短语可以出现在消息的任何位置，查询从触发短语开始并扩展到“最大单词数”。
  4. 正则表达式 - 提供一个JS风格的正则表达式来匹配用户消息。如果正则表达式匹配，搜索将以给定的查询进行触发。搜索查询支持{{macros}}和$1语法来引用匹配的组。例如，/what is happening in (.*)/i正则表达式会匹配包含what is happening in New York的消息，并触发查询 news in New York。
  5. 触发短语 - 添加一个个短语来触发搜索，可以是消息中的任何位置，查询将从触发词开始并扩展到“最大单词数”。要排除特定消息不被处理，必须以句点（.）开头，例如.你怎么认为？。触发短语的优先级：首先按文本框中的顺序，然后按用户消息中的第一个短语。
  6. 最大单词数 - 包括在搜索查询中的单词数量（包括触发短语）。Google的查询限制大约是32个单词。默认值 = 10个单词。

页面抓取

  1. 访问链接 - 将从访问的搜索结果页面中提取文本并保存为文件附件。
  2. 访问次数 - 将访问并解析多少个链接的文本。
  3. 访问域名黑名单 - 排除访问的站点域名。每个域名一行。
  4. 文件头 - 文件头模板，在文本文件的开始插入，可额外使用{{query}}宏。
  5. 区块头 - 链接区块模板，插入每个链接的解析内容。使用{{link}}宏表示页面URL，使用{{text}}宏表示页面内容。
  6. 保存目标 - 保存抓取结果的目标。可能的选项：触发的消息附件，或者聊天附件的“数据仓库”。

更多信息

从最新查询中获得的搜索结果将在下一个有效查询找到之前一直包含在提示词中。如果你想在没有意外触发搜索的情况下提问额外问题，可以以句点开头。

Note

如果启用了Web搜索功能工具，它会覆盖其他触发方式。

触发优先级（如果启用了多个触发器）：

  1. 反引号。
  2. 正则表达式。
  3. 触发短语。

要丢弃所有之前的查询进行处理，可以以感叹号开头，如!现在我们来聊聊...，这将丢弃当前及其上方的所有消息。

此扩展还提供了一个/websearch斜杠命令，可用于ST脚本。更多信息请见：ST脚本语言参考

/websearch links=on|off snippets=on|off [query] – 执行一个网页搜索查询。使用命名参数指定返回内容 - 页面片段（默认：开启），完整页面内容（默认：关闭），或两者。

示例：/websearch links=off snippets=on 如何做三明治

搜索结果中可以包含哪些内容？

SerpApi

  1. 答案框。问题的直接答案。
  2. 知识图谱。关于该主题的百科知识。
  3. 页面片段（最多10个）。来自网页的相关摘录。
  4. 相关问题（最多10个）。与类似话题相关的问题和答案。

Selenium 插件和Extras API

  1. Google - 答案框，知识图谱，页面片段。
  2. DuckDuckGo - 页面片段。

SearXNG

  1. 信息框。
  2. 页面片段。

Tavily AI

  1. 答案。
  2. 页面内容。
